# 📱 Advanced Android Calculator

An advanced calculator app built using Android (Java).

## ✨ Features
- Basic Operations (+, -, ×, ÷)
- Power (^)
- Square Root (√)
- Percentage (%)
- Dark Theme UI
- Error Handling

## 🛠 Built With
- Java
- Android Studio
- Gradle

## 📂 Project Structure
app/
 ├── src/
 ├── res/
 └── AndroidManifest.xml
